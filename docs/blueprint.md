# **App Name**: OwareAds

## Core Features:

- Billboard Browsing on Interactive Map: Allow businesses to browse available billboards on an interactive map of Ghana using Leaflet and OpenStreetMap.
- Billboard Rental and LED Slot Booking: Enable businesses to rent billboard spaces and book digital LED slots directly through the platform.
- Vendor Marketplace: Connect businesses with printing vendors, graphic designers, and installation technicians.
- Mobile Money Payment Integration: Facilitate payments via Mobile Money and card through Paystack or Flutterwave.
- Admin-Controlled Billboard Pricing: Allow administrators to manually set and control pricing for billboards, including seasonal overrides and bulk discounts.
- Ghana-Specific Features: Implement Ghana-specific features such as WhatsApp inquiry button per billboard, installment booking options, and SME starter packages.
- AI-Powered Location and Content Recommendations: Use AI as a tool to provide campaign location recommendations and similar billboard suggestions.  This AI will provide the admin suggested ranges for setting billboard pricing manually. Pricing must be strictly set by Admin and NO automatic pricing is allowed.

## Style Guidelines:

- Primary color: Vibrant orange (#FF9933) to reflect the energy and vibrancy of Ghana's advertising market.
- Background color: Light beige (#F5F5DC), offering a neutral backdrop that emphasizes the visual content.
- Accent color: Dark green (#38761D) to complement the orange, suggestive of growth and sustainability within the Ghanaian economy.
- Headline font: 'Belleza', a humanist sans-serif for its style aligned to fashion, art, and design
- Body font: 'Alegreya', a humanist serif, for longer texts needing elegance and readability
- Use clean, modern icons to represent different billboard types, services, and vendors.
- Prioritize a clean, intuitive layout with a focus on map integration and billboard browsing. Utilize marker clustering to manage density and improve performance.
